# Machine Learning-Classification

Overview:
This project predict is about build a classification model that predicts whether a person is diabetic or not

Technology Used:
• Python
• Numpy
• Pandas
• Matplotlib
• Scikit learn
• Google colab

Algoritham Used:
• KNN

Model Performance:
• Accuracy score = 74%

Columns in dataset:
• Pregnancies
• Glucose
• BloodPressure
• SkinThickness
• Insulin
• BMI	
• DiabetesPedigreeFunction
• Age
• Outcome (Output Column)

Work Flow:
1.Import the required libraries
2.Load and explore the dataset
3.Preprocess the data
4.Split the dataset into training and testing sets
5.Train the KNN classifier
6.Evaluate the model using accuracy
7.Predict whether a person is diabetic
